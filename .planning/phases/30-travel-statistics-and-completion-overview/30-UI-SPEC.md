## Spacing Scale

Declared values (must be multiples of 4):

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px (var(--space-xs)) | Icon gaps, inline padding |
| sm | 8px (var(--space-sm)) | Compact element spacing, badge padding |
| md | 16px (var(--space-md)) | Default element spacing, card inner padding |
| lg | 24px (var(--space-lg)) | Section padding, card gap |
| xl | 32px (var(--space-xl)) | Layout gaps |
| 2xl | 48px (var(--space-2xl)) | Major section breaks |
| 3xl | 64px (var(--space-3xl)) | Page-level spacing |

Exceptions: stat number display uses 20px (p-5) inner padding matching timeline card pattern. Touch targets for nav buttons minimum 44px height (min-h-11).

### StatCard Layout Contract

Each `StatCard` displays one metric:

```
┌─────────────────────────────────────────┐
│  [EYEBROW LABEL — 0.72rem uppercase]    │
│                                         │
│  [LARGE NUMBER — clamp(2rem,3.5vw,3rem)]│
│  [UNIT LABEL — 14px muted]              │
└─────────────────────────────────────────┘
```

- Border radius: `rounded-[28px]` (var(--radius-card))
- Border: `border border-white/80`
- Shadow: `shadow-[var(--shadow-float)]`
- Padding: `p-5` (20px)
- Two cards side by side on md+: `grid grid-cols-1 md:grid-cols-2 gap-4 md:gap-5`

### Two Stat Cards

1. "总旅行次数" card — gradient: `linear-gradient(180deg,rgba(255,255,255,0.94),rgba(255,246,250,0.94))`
2. "已去过地点数" card — gradient: `linear-gradient(180deg,rgba(255,255,255,0.92),rgba(239,250,252,0.94))`

Source: D-06 from CONTEXT.md (metrics scope), TimelineVisitCard.vue pattern.

## Interaction States

| State | Trigger | Visual |
|-------|---------|--------|
| restoring | auth status === 'restoring' | Skeleton shimmer cards (white/80–90 rounded-full placeholders) |
| anonymous | not authenticated | Anonymous panel with "立即登录" accent CTA button |
| empty | authenticated, 0 records | Empty panel with "去地图添加旅行" ghost pill button |
| populated | authenticated, records exist | Two StatCards in grid |
| error | API /stats fetch fails | Error panel: problem description + retry link |

State flow mirrors `TimelinePageView.vue` exactly — same 4-state pattern (isRestoring / shouldShowAnonymousState / shouldShowEmptyState / shouldShowStats).

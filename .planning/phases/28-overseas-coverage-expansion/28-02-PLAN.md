## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| client payload -> `/records` / `/records/import` | 不可信的海外旅行记录 metadata 跨入 authoritative server write path。 |
| generated manifest -> backfill script -> database | build 产出的 canonical metadata 会覆盖已持久化记录。 |
| database -> `/auth/bootstrap` | persisted metadata 被直接回放到 web consumer，不会在客户端重新推导。 |

| Option | Description | Selected |
|--------|-------------|----------|
| 优先打开现有记录 | 不创建重复草稿 | ✓ |
| 先提示再选择打开旧记录或继续新建 | 更谨慎但多一步 | |

**User's choice:** 优先打开现有记录  
**Notes:** 这适用于再次点击已记录城市的默认行为。

| Option | Description | Selected |
|--------|-------------|----------|
| 稳定城市身份 ID | 作为“同一城市”的主键 | ✓ |

**User's choice:** 稳定城市身份 ID  
**Notes:** 不依赖 `cityName` 文本，方便后续边界高亮和持久化恢复。

| Option | Description | Selected |
|--------|-------------|----------|
| 搜索和点击都统一复用 | 入口不同，规则一致 | ✓ |

**User's choice:** 搜索和点击都统一复用  
**Notes:** 不希望因为入口不同而得到不同的重复处理规则。

| Option | Description | Selected |
|--------|-------------|----------|
| 给一句轻提示 | 说明发生了复用，但不打断 | ✓ |

**Notes:** 示例方向为“已打开你记录过的 Kyoto”。

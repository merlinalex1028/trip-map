<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| anonymous map interaction → authenticated save path | 未登录用户可能直接触发保存动作 |
| app-level import dialog → store actions | UI 需要忠实反映 store 决策，不得自己发明第三条路径 |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-24-07 | Elevation of Privilege | anonymous save interception | mitigate | 匿名态保存只打开登录弹层，不触发任何 records 写入 |
| T-24-08 | Information Disclosure | map context continuity | mitigate | 拦截保存时保留当前 summary surface、地图位置与识别结果，不把用户打回空白初始态 |
| T-24-09 | Repudiation | import choice UX | mitigate | dialog 只提供两条显式路径，并显示 authoritative summary，避免系统静默替用户决定 |
</threat_model>

<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| browser localStorage → auth-session store | legacy snapshot 可能损坏、过期或伪造 |
| auth-session import gate → map-points records snapshot | 迁移失败或 cloud-wins 分支可能误清空当前权威记录 |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-24-04 | Tampering | legacy local snapshot parsing | mitigate | 只接受 `version: 2` 且结构完整的 snapshot；`corrupt/incompatible` 走失败分支，不触发导入 |
| T-24-05 | Denial of Service | repeated migration prompt | mitigate | cloud-wins 与 import 成功后清理 `trip-map:point-state:v2`，避免每次登录都重复打断用户 |
| T-24-06 | Information Disclosure | import gate state vs live records | mitigate | legacy 检测发生在 authenticated snapshot 之后，且 cloud snapshot 保持 authoritative，不因本地旧记录检测而被清空 |
</threat_model>

## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| foreground bootstrap snapshot → local mutation state | 同账号前台刷新返回的是权威快照，但本地可能正处于 optimistic mutation 中 |
| mutation result → user-facing notice | 成功、普通失败、会话失效如果被并发逻辑混淆，用户会误判当前账号状态 |

Create all three composables and their spec files. Specs mock `leaflet` module entirely (vi.mock('leaflet')) to avoid DOM/Canvas issues in happy-dom. Use the patterns from RESEARCH.md.

**useLeafletMap.ts:**
- Import `L` from 'leaflet' and Vue lifecycle hooks
- Accept `containerRef: Ref<HTMLElement | null>` parameter
- In onMounted: create `L.map(containerRef.value, { center: [35.0, 105.0], zoom: 4, minZoom: 2, maxZoom: 10, zoomControl: true })`, then add tile layer via `addTileLayer(map)` helper
- `addTileLayer`: check `import.meta.env.VITE_BING_MAPS_KEY`. If present, call `createBingTileLayer(key)` which fetches `https://dev.virtualearth.net/REST/V1/Imagery/Metadata/CanvasLight?output=json&uriScheme=https&culture=zh-CN&key=${key}`, parses the response, builds URL template replacing `{subdomain}`->`{s}`, `{zoom}`->`{z}`, `{x}`->`{x}`, `{y}`->`{y}`, creates `L.tileLayer(template, { subdomains, attribution: '© Microsoft Bing Maps', maxZoom: 19 })`. If fetch fails or key absent, fall back to `L.tileLayer('https://{s}.basemaps.cartocdn.com/light_all/{z}/{x}/{y}{r}.png', { attribution: '© OpenStreetMap contributors © CARTO', maxZoom: 19, subdomains: 'abcd' })`
- Set `isReady.value = true` after tile layer added
- Use `shallowRef` for map instance (not reactive proxy — Leaflet objects are not plain objects)
- In onBeforeUnmount: `map.value?.remove(); map.value = null`
- Use ResizeObserver on container to call `map.value?.invalidateSize()`

**useGeoJsonLayers.ts:**
- Accept `{ map: ShallowRef<L.Map | null>, savedBoundaryIds: Ref<string[]>, selectedBoundaryId: Ref<string | null>, onBoundaryClick: (boundaryId: string, latlng: L.LatLng) => void }`
- Create `cnLayer = L.geoJSON(undefined, { style: styleFunction, onEachFeature: featureClickHandler })` and `overseasLayer` similarly
- `styleFunction(feature)`: read `feature.properties.boundaryId`, compare against `selectedBoundaryId.value` and `savedBoundaryIds.value` to return correct PathOptions (exact color values from UI-SPEC.md state colors table)
- `featureClickHandler(feature, layer)`: `layer.on('click', (e) => { L.DomEvent.stopPropagation(e); onBoundaryClick(feature.properties.boundaryId, e.latlng) })`
- `addFeatures(target: 'CN' | 'OVERSEAS', fc: CityBoundaryFeatureCollection)`: call `targetLayer.addData(fc)`
- Watch `savedBoundaryIds` and `selectedBoundaryId`: on change, call `cnLayer.setStyle(styleFunction)` and `overseasLayer.setStyle(styleFunction)` to refresh all feature styles
- Watch `map`: when map becomes available, add both layers to map
- Cleanup: remove layers from map on unmount
- Export: `{ cnLayer, overseasLayer, addFeatures, refreshStyles }`

**useLeafletPopupAnchor.ts:**
- Accept `{ map: ShallowRef<L.Map | null>, latlng: Ref<L.LatLng | null>, onPositionUpdate: () => void }`
- Compute `virtualElement: ComputedRef<VirtualElement | null>` — when map and latlng are both non-null, return object with `getBoundingClientRect()` that calls `map.latLngToContainerPoint(latlng)`, adds `map.getContainer().getBoundingClientRect()` offsets, returns DOMRect
- Watch map: when available, `map.on('move zoom', onPositionUpdate)` — cleans up old listeners when map changes
- On unmount: remove event listeners
- Export: `{ virtualElement }`

## Interaction Contract

### Map click flow

1. User clicks tile base map (not a saved boundary).
2. Pending `L.circleMarker` appears at click coordinates. Style: radius 8, fill `rgba(244, 143, 177, 0.94)`, stroke `rgba(244, 143, 177, 0.96)` weight 1.5, CSS class `pending-marker--recognizing`.
3. Canonical resolve API called with `{ lat, lng }`.
4. On success: circleMarker removed; popup appears anchored via @floating-ui VirtualElement at the click LatLng converted to container point. GeoJSON layer highlight applied.
5. On failure: circleMarker removed; inline error notice shown at map level (small toast or inline panel — Claude discretion).

### Boundary click (already-saved place)

1. User clicks a GeoJSON feature whose `boundaryId` is in `savedBoundaryIds`.
2. `L.DomEvent.stopPropagation(e)` prevents map-level click handler.
3. Popup opens directly with the existing saved-place surface — no server resolve call.
4. Selected boundary style applied immediately.

### Popup anchor behavior

- Anchor: `@floating-ui` VirtualElement computed from `map.latLngToContainerPoint(latlng)` + container offset.
- Refresh: `map.on('move zoom', updatePosition)` keeps popup synchronized during pan and zoom.
- Arrow: visual arrow pointing downward from popup base, CSS rotated square (existing `.map-context-popup__arrow`).
- Popup width: min 280px, max 360px (existing `--map-context-popup-min-width/max-width`).

### GeoJSON highlight state machine

Three exclusive states per feature:

| State | Trigger | Style |
|-------|---------|-------|
| unrecorded | default | opacity 0, fillOpacity 0 (invisible) |
| saved | `boundaryId` in `savedBoundaryIds` | stroke `rgba(132, 199, 216, 0.82)` weight 2.4, fill `rgba(132, 199, 216, 0.24)` |
| selected | `boundaryId === selectedBoundaryId` | stroke `rgba(244, 143, 177, 0.96)` weight 3.2, fill `rgba(244, 143, 177, 0.28)` |

Selected overrides saved if a saved boundary is also currently selected.

Style refresh: `watch(savedBoundaryIds, () => layer.setStyle(styleFunction))` + `watch(selectedBoundaryId, () => layer.setStyle(styleFunction))`.

Single `currentSelectedLayer` reference maintained in `useGeoJsonLayers.ts` to prevent double-highlight (MAP-08).

### Map initialization parameters

| Parameter | Value | Rationale |
|-----------|-------|-----------|
| Initial center | `[35.0, 105.0]` | China-centered (D-03) |
| Initial zoom | 4 | China fills most of viewport at zoom 4 |
| minZoom | 2 | Full world visible; boundaries still distinguishable |
| maxZoom | 10 | City-level detail; beyond this, admin-1 boundaries lose meaning |
| zoomControl | true | Default Leaflet control, top-left position |

Source: D-04 (zoom limits), D-03 (China center). Specific values are Claude's discretion.

### Tile base map

Primary: Bing Maps CanvasLight via REST Imagery Metadata API. URL parameters: `culture=zh-CN` for Chinese-first labels, `uriScheme=https`.

Fallback (when `VITE_BING_MAPS_KEY` is absent or metadata fetch fails): CartoDB Positron (`https://{s}.basemaps.cartocdn.com/light_all/{z}/{x}/{y}{r}.png`). Attribution: `© OpenStreetMap contributors © CARTO`.

Both options are visually pale/desaturated — no CSS filter needed (D-05 satisfied by tile style choice).

## Motion Contract

Carry forward from `tokens.css`:

| Token | Value | Usage |
|-------|-------|-------|
| `--motion-quick` | 140ms | Button hover/active transitions |
| `--motion-emphasis` | 180ms | Popup entrance, boundary highlight state change |

GeoJSON layer style transitions: CSS `transition` on `.leaflet-interactive` elements for fill/stroke changes. Use `--motion-emphasis` (180ms) ease-out.

Pending marker: no entrance animation — appears immediately on click, removed immediately on resolve completion.

Popup: CSS `opacity` + `transform: translateY` entrance at `--motion-emphasis` 180ms ease-out.

## Accessibility Contract

- Popup `<aside>` with `role="dialog"`, `aria-modal="false"`, `aria-labelledby` pointing to visually-hidden `<h2>`. Carry forward unchanged from `MapContextPopup.vue`.
- All interactive buttons: `min-height: 44px` touch target.
- Pending marker: add `aria-live="polite"` region on the map container to announce "正在识别…" and resolve result to screen readers.
- Leaflet map container: `aria-label="旅行世界地图"` on the map div.
- Focus management: on popup open, focus moves to popup title element (`tabindex="-1"` h2). Carry forward from existing `focusEntryPoint()` pattern.
- Focus return: on popup close, return focus to the map container.

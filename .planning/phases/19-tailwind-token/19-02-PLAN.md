<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| global CSS entry → Leaflet third-party CSS | Tailwind preflight 会先改写全局元素默认样式，再由 Leaflet CSS 和项目兼容层恢复地图控件 |
| app shell template → Leaflet map stage | App 壳层 class 直接包裹 Leaflet 舞台，任何 transform/filter 都可能影响地图坐标和控件表现 |

</threat_model>

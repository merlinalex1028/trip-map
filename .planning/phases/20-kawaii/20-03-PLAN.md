## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| popup content text → rendered card UI | 标题、notice、candidate hint 都来自不受信文本，需要保持安全插值 |
| decorative cloud layers → CTA / candidate hit areas | 伪元素或重叠阴影若拦截点击，会直接破坏按钮可用性 |
| card motion classes → Leaflet map geometry | 若把 motion 误扩散到地图容器或全局选择器，会造成命中漂移 |

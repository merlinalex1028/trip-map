  <name>Task 2: 修复地图偏移与顶部间隙——布局层调整</name>
  <files>apps/web/src/App.vue, apps/web/src/components/LeafletMapStage.vue</files>
  <action>
**App.vue — poster-shell__experience**

当前：
```css
.poster-shell__experience {
  display: grid;
  gap: var(--space-md);
  padding: var(--space-md);
  ...
}
```

修改为：
```css
.poster-shell__experience {
  display: flex;
  flex-direction: column;
  padding: 0;          /* 去掉 padding，让地图贴卡片内边缘 */
  gap: 0;              /* 去掉 gap */
  ...
}
```
其余属性（border, border-radius, background, box-shadow, overflow: hidden, isolation）保持不变。

`::before` 和 `::after` 伪元素保持不变（含 dashed 内边框装饰）。

**App.vue — poster-shell__stage**

当前：
```css
.poster-shell__stage {
  grid-area: stage;
  min-height: 68vh;
  position: relative;
  z-index: 1;
}
```

修改为：
```css
.poster-shell__stage {
  flex: 1 1 0;         /* 在 flex 容器中撑满剩余高度 */
  min-height: 0;
  position: relative;
  z-index: 1;
}
```

**App.vue — poster-shell（外层 grid）**

当前 poster-shell 的 `grid-template-rows: auto minmax(0, 1fr)` 已经让 stage 区域占满剩余高度，无需修改。但需要让 `.poster-shell__experience` 本身具有明确高度：当 `poster-shell` 的 stage grid area 撑满时，experience 作为该 area 中的 flex 容器，需要 `height: 100%`：

在 `.poster-shell__experience` 中添加 `height: 100%`。

**LeafletMapStage.vue — 让地图填满 stage**

`.leaflet-map-stage`：
- 去掉 `min-height: 68vh`
- 改为 `height: 100%; display: flex; flex-direction: column;`

`.leaflet-map-stage__map`：
- 将 `height: 68vh; min-height: 400px` 改为 `flex: 1 1 0; min-height: 0`
- 保留 `width: 100%`
- 保留 `border-radius: 28px; overflow: hidden; border; background; z-index: 0`

这样 Leaflet 容器的实际像素高度由 flex 布局决定，ResizeObserver 会在尺寸确定后触发 `invalidateSize()`，修复地图只显示右半的问题。

## Phase 15 Gap Analysis (Critical Path)

Phase 15 historically introduced the two requirements that were previously marked partial (API-01, API-02). The UAT in `15-UAT.md` diagnosed two major gaps:

### Gap 1: 点亮 button silent failure on fallback points
- **Root cause**: `handleIlluminate` in `LeafletMapStage.vue` had a silent guard `if (!point.placeId || !point.datasetVersion) return;` — fallback draft points (OUTSIDE_SUPPORTED_DATA path) have these fields as null
- **Impact**: Button appeared clickable but did nothing for fallback points
- **Phase 16-02 fix**: Added `isIlluminatable` prop to disable button on fallback points; replaced silent guard with `setInteractionNotice` feedback
- **Status**: ✅ Code-level fix committed

### Gap 2: SmokeRecord missing 5 canonical metadata columns
- **Root cause**: Prisma `SmokeRecord` model lacked `regionSystem`, `adminType`, `typeLabel`, `parentLabel`, `subtitle` columns
- **Impact**: `createSmokeRecord()` in repository could not write these fields; `records-smoke.e2e-spec.ts` failing
- **Phase 16-01 fix**: Added nullable columns via migration, backfilled existing records, updated service/repository
- **Status**: ✅ Schema + backfill committed

### Closure
Phase 17 已补齐 `15-VERIFICATION.md`，并通过 `16-HUMAN-UAT.md` 的 3/3 pass 把 fallback 点亮反馈、saved overlay 可见性、California 标签一致性都提升为正式闭环证据。
